Create a Payments layer so providers are swappable.

DB: payment table
	•	id PK
	•	booking_id FK NULL (for hourly bookings)
	•	pass_id FK NULL (for passes)
	•	provider TEXT CHECK(mercury,chase,manual)
	•	intent_id TEXT (Mercury invoice ID / Chase txn ID)
	•	status TEXT CHECK(created,pending,paid,failed,refunded)
	•	amount_cents INT
	•	currency TEXT DEFAULT usd
	•	created_at DATETIME

2) Mercury (online)

	•	If Mercury doesn’t provide webhooks in your plan, implement polling (cron every 2–5 min) or manual “Mark Paid” button in Admin.

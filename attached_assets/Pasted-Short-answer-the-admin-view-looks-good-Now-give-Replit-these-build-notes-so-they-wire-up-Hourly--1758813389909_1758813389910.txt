1) UI flow

Entry point: “Book Desk”
	1.	Modal Step 1: Choose resource (Hot Desk / Quiet Desk / Meeting Lounge / Whole Room).
	2.	Modal Step 2: Choose plan → Hourly | Day Pass | Week Pass | Month Pass.
	3.	Modal Step 3: Plan-specific form (see below).
	4.	Availability check → price summary → pay (Mercury) or mark POS (Chase) → success.
	5.	On success, create booking(s) and block the calendar.

Forms per plan
	•	Hourly
	•	Date
	•	Start time, End time (respect opening hours)
	•	Seats (<= capacity)
	•	Day Pass
	•	Date (single day)
	•	Seats (<= capacity)
	•	If resource has business hours, booking blocks that day within hours.
	•	Week Pass
	•	Start date (blocks 7 consecutive days within business hours)
	•	Seats (<= capacity)
	•	Month Pass
	•	Start date (blocks 30 consecutive days within business hours)
	•	Seats (<= capacity)

Important: For Week/Month, we are reserving the same specific desk/room for the duration. That means capacity for that resource is reduced for all overlapping hours in that period.

4) Availability engine (no conflicts)

A booking is allowed only if seats_requested ≤ seats_available for every hour in the requested window.

-- seats already booked for a resource during a given window:
SELECT COALESCE(SUM(seats), 0) AS booked
FROM bookings
WHERE resource_id = :rid
  AND status IN ('pending','paid')
  AND NOT (end_dt <= :new_start OR start_dt >= :new_end);

Seats available = resource.capacity - booked. Must be ≥ seats_requested.

Hourly: start_dt = selected date+time; end_dt = same date + hours.
Day/Week/Month:
	•	Create one continuous booking per pass: start_dt = start date 09:00, end_dt = end date 17:00 (or whatever opening hours JSON defines daily).
	•	For availability, check every open hour in the range; if any hour would exceed capacity, block with a clear message.

Edge cases
	•	Validate against opening_hours (reject out-of-hours)
	•	Enforce timezone (store UTC, display local ET)
	•	Atomicity: on payment webhook success, finalize booking; if payment fails/timeout, auto-cancel pending holds (e.g., after 15 minutes).
